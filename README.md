
🎓 Campus Administration Suite (Angular & Django)
A high-performance management system demonstrating a professional decoupled architecture using Angular and Django.
🛠️ Technical Stack
Frontend: Angular (TypeScript), RxJS, Bootstrap.  
Backend: Django 3.2, Django REST Framework (DRF).  
Database: MySQL with XAMPP integration.  
Tools: VS Code, Postman for API testing.  
🚀 Core Functionalities
Enterprise CRUD: Advanced student record management with real-time updates.  
Type-Safe Frontend: Utilized TypeScript for robust frontend logic and data handling.  
Cross-Origin Support: Implemented CORS middleware for secure communication between Angular (Port 4200) and Django (Port 8000).
